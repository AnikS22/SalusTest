   Error: AppLauncher hangs forever
   Cause: Importing torch/numpy before AppLauncher
   Solution: Only import argparse/sys/Path before AppLauncher

Fix #3: Duplicate AppLauncher
   Error: Hang during environment import
   Cause: Environment file created second AppLauncher
   Solution: Remove from environment, pass as parameter

Fix #4: Ground Plane Config
   Error: Unknown asset config type
   Cause: Isaac Lab 0.48.5 API change
   Solution: Wrap GroundPlaneCfg in AssetBaseCfg

   Error: 'Articulation' object has no attribute 'actuators'
   Cause: scene.reset() called before sim initialized
   Solution: Call sim.reset() in first env.reset()

Fix #6: Action Dimensions
   Error: Shape mismatch [7] vs [1, 9]
   Cause: VLA outputs 7 DOF, Franka has 9 joints
   Solution: Pad actions with gripper commands

Fix #7: GPU Device Mismatch
   Error: Tensors on cuda:1 and cuda:0
   Cause: VLA on cuda:0, Isaac on cuda:1
   Solution: Run everything on cuda:0 with CUDA_VISIBLE_DEVICES=0
